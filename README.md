## 何ができるか ##

Gmailやサイトなどに画像ファイルとして埋め込むと、スプレッドシートでアクセス解析（時間、アクセスしたメールアドレス）できる。

## 運用方法 ##

Googleドライブにこのリポジトリと同じファイルを作成する。

スプレッドシートに、アクセス解析する「タグ」、「ID」をそれぞれ「シート名」、「e1セル」に書き込む。

の形式で埋め込むと、アクセス解析ができるようになる。
